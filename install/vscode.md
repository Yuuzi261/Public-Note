# 安裝Visual Studio Code

## 下載Visual Studio Code

1. 搜尋visual studio code / vscode </br></br>
2. 進入官網 </br></br>
3. 點選Download
![](https://i.imgur.com/FGdaU9L.png)

## 開始安裝

![](https://i.imgur.com/6iW4k2b.png) </br></br>
